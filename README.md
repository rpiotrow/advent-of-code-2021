# Solutions for Advent of Code 2021 puzzles

Puzzles available at https://adventofcode.com/2021/

## Run

Run all:
```
$ sbt run
```

Run selected day:
```
$ sbt run -d <day_number>
```

## Tests

Run unit tests:
```
$ sbt test
```

Run regression tests (check solution answers):
```
$ sbt regression:test
```

## Used libraries
